MVP Acadêmico — sem banco de dados
O DataLoader carrega os 3 arquivos JSON em memória na inicialização
via @PostConstruct. Não há Spring Data JPA nem @Entity.

V1 (produto real): PostgreSQL + Spring Data JPA + Spring Security + JWT
